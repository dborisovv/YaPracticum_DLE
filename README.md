### Здесь хранятся учебные проекты из курса Яндекс Практикума "Инженер глубокого обучения":
- sprint2project - построение нейронной сети архитектуры LSTM для генерации автодополонений коротких текстов. Сравнение с автодополонениями, сгенерированными трансформером. 
    - /data: иcходный датасет для обучения (raw_dataset), содержащий более полутора миллионов твитов на английском языке. Также обработанные тексты, тренировочная, валидационная и тестовая выборки в формате PyTorch Dataset.
    - /src: исходный код обработки текстов, кастомного датасета, LSTM модели, обучения модели, оценки генераций модели метрикой ROUGE. Также оценка автодополнений, сгенерированных трансформером.
    - /models: веса обученной LSTM.
    - solution.ipynb: процесс подготовки данных, обучения модели, оценки генераций.
    - lstm_preds.txt, gpt_preds.txt - прмиеры автодополнений, сгенерированных LSTM и трансформером (входной текст и генерация разделены вот так --).
    - requirements.txt: зависимости проекта.
    - training_process.txt: вывод процесса обучения - лоссы, метрики, примеры генераций по эпохам.
